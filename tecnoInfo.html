Fundamentos da Tecnologia da Informação

Hardware

Hardware é a parte física do computador, ou seja, todos os componentes que podemos tocar, como teclado, monitor, HD, memória e placas internas.

Placa Mãe

É a principal placa do computador, onde todos os outros componentes são conectados (processador, memória, HD, etc.). Ela permite que as peças se comuniquem entre si.

Processador

É o cérebro do computador. Responsável por executar os cálculos e instruções dos programas. Quanto mais rápido o processador, melhor o desempenho da máquina.

Memória Principal

É a memória temporária usada enquanto o computador está ligado. Armazena dados e programas em uso, permitindo acesso rápido pelo processador. Ao desligar o computador, tudo o que está na RAM é apagado.

Sistema Operacional

Um sistema operacional (SO) é o programa principal do computador, responsável por controlar o hardware e permitir que o usuário use o computador e execute outros programas. Ele faz a ponte entre o usuário e as partes físicas da máquina (como processador, memória e disco).

Exemplos

  • Windows
  • Linux
  • macOs
  • Android

    • Sistema de Arquivos

    O sistema de arquivos é o método que o sistema operacional usa para organizar e armazenar os dados no disco do computador (HD, SSD, pen drive etc.). Ele define como os arquivos e pastas são criados, nomeados, acessados e apagados.

    Modo Texto (CMD ou terminal)

    O usuário digita comandos para navegar e gerenciar arquivos. É mais técnico, mas rápido e preciso para quem sabe os comandos.

    Modo Gráfico (Explorer ou Gerenciador de Arquivos)

    Usa ícones, janelas e cliques do mouse para manipular os arquivos. O usuário pode abrir, mover, copiar ou excluir arquivos de forma visual. É mais fácil e intuitivo, ideal para o uso diário.

    Software de Escritório

    Software de escritório é um tipo de programa usado para tarefas do dia a dia em escritórios, escolas e empresas, como escrever textos, criar planilhas, fazer apresentações e organizar informações.

    Exemplos

  • Microsoft Word
  • Microsoft Excel
  • Microsoft PowerPoint
