1. O que são Linguagens de Programação?

Linguagens de programação são conjuntos de regras e comandos usados para criar programas. Com elas, desenvolvedores conseguem orientar o computador a executar tarefas e transformar ideias em softwares, jogos e sites.

  • C++

    Linguagem compilada, rápida e usada em jogos, sistemas operacionais e aplicações que exigem alto desempenho.

  • JavaScript

    Usada na criação de páginas web interativas e também no back-end com Node.js.

  • Java

    Multiplataforma e orientada a objetos. Muito usada em apps Android e sistemas corporativos.

  • Python

    Linguagem simples e versátil, usada em IA, ciência de dados, automação e educação.

  • PHP

    Voltada ao desenvolvimento web e bastante utilizada em sites e servidores, como no WordPress.

2. Interpretada vs Compilada

Interpretada: o código é executado linha por linha, sem gerar um arquivo final.

Compilada: o código é convertido antes em um executável, geralmente garantindo mais velocidade.

3. O que é HTML e CSS?

HTML: estrutura o conteúdo de uma página.

CSS: define o estilo visual, como cores, fontes e layout.

4. O que é uma IDE?

Uma IDE é um programa que reúne ferramentas para escrever, testar e corrigir códigos em um único lugar.

Exemplos: VSCode, IntelliJ, Eclipse.
